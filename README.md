## Laravel Space App

Na pasta backend encontra-se a API <br>
Os testes (Postman) da api estão em:
```
backend/tests/LaravelSpaceApp.postman_collection.json
```
* Instalação após baixar
```bash
composer install
```
* Migrations
```bash
php artisan migrate
```
* Seeders
```bash
php artisan db:seed
```
* Server
```bash
php artisan serve
```
